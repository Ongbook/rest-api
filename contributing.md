# API de Dados do Ongbook.org

Este projeto foi desenvolvido utilizando o ASP.NET Core.

## Instalação do dotnet core
Você pode efetuar a instalação para Windows, Linux e Mac. Para mais detalhes e integrações como Visual Studio e Visual Studio Code consulte https://www.microsoft.com/net/core

## Instalação do projeto
```sh
$ git clone https://github.com/Ongbook/rest-api.git
```
para inicializar o projeto e baixar suas depêndências use
```sh
$ dotnet restore
```
esse comando baixa todas as dependências do dotnet core assim como as dependências listadas pelo bower e npm.

#### Testando a aplicação
para iniciar o servidor e rodar a aplicação use
```sh
$ dotnet run
```
